Paper digest -- Matching networks for few-shot learning

Cong Wang

Jun 14, 2022

The paper

  • Matching networks for one-shot learning (Vinyals et al. 2017).

    Use an attention mechanism over a learned embedding of the labeled set of examples (the support set) to predict classes for the unlabeled points (the query set).

Matching networks for one-shot learning | Formulate the problem

Given

  • A support set \(\mathbb{S} \equiv \{(\boldsymbol{x} _1, \boldsymbol{y} _1), \cdots, (\boldsymbol{x} _S, \boldsymbol{y} _S)\}\), whose classes are not seen by the model below.

  • A trained image-to-embedding model: \(\mathit{pretrainModel} _{\boldsymbol{\Theta} }(\boldsymbol{x} ) : \mathbb{R}^H \times \mathbb{R}^W \mapsto \mathbb{R}^N\)

    …, \(\boldsymbol{x} \in \mathbb{X} \equiv \{\boldsymbol{x} _{1},\cdots,\boldsymbol{x} _{K}\}\), a set of training images.

    …, \(\boldsymbol{\Theta}\) represents the set of parameters in the model \(\mathit{pretrainModel}\).

  • An unlabeled query set \(\mathbb{T} \equiv \{\hat{\boldsymbol{x}}_1, \cdots, \hat{\boldsymbol{x}}_T\}\).

Find a model \(h_{} : \mathbb{R}^H \times \mathbb{R}^W \mapsto \mathbb{N}\) that produces an integer label \(\hat{\boldsymbol{y}}= h_{}(\hat{\boldsymbol{x}}\mid \mathbb{S} )\).

 

Remarks: The model \(h\) doesn’t have intrinsic/structural parameters. \(\boldsymbol{\Theta}\) is only a byproduct of using a pretrained artificial neural network model.

Matching networks for one-shot learning | What’s the model \(h\)?

Matching networks for one-shot learning | Training strategy

Suppose \(\mathit{UL(\mathbb{X} )} \equiv \mathit{uniqueLabel(\mathbb{X} )}\) returns a set of unique labels in a training set \(\mathbb{X} \equiv \{ (\boldsymbol{x} _i, \boldsymbol{y} _i) \mid i = 1, \cdots, X \}\).

 

Given a training set \(\mathbb{X} \equiv \{ (\boldsymbol{x} _i, \boldsymbol{y} _i) \mid i = 1, \cdots, X \}\)

…, sample a subset \(\mathcal{L} \sim \mathit{UL}(\mathbb{X} )\) resulting in a subset of all unique labels.

…, meanwhile, \(\mathbb{L} \equiv \{ (\boldsymbol{x} _i, \boldsymbol{y} _i) \mid (\boldsymbol{x} _i, \boldsymbol{y} _i) \in \mathbb{X} \land \boldsymbol{y} _i \in \mathcal{L} \}\) which is the set of all examples with labels belonged to \(\mathcal{L}\).

…, sample a subset \(\mathbb{S} \sim \mathbb{L}\) as the support set.

…, sample a subset \(\mathbb{Q} \sim \mathbb{L}\) as the query set.

 

\[\boldsymbol{\Theta} = \arg\max_{\boldsymbol{\Theta} } \sum_{(\hat{\boldsymbol{x}}, \hat{\boldsymbol{y}})\in \mathbb{Q}} \log h_{}( \hat{\boldsymbol{x}}\mid \mathbb{S} ))\]

…, but where the error is? Is \(\hat{\boldsymbol{y}}\) supposed to appear in the model \(h\)?
